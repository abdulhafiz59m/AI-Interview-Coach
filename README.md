# AI Interview Coach 🎯

AI Interview Coach is a smart interview preparation tool that generates role-specific interview questions and provides instant feedback on candidate responses. The application uses Gemini AI for dynamic question generation and includes a fallback question bank to ensure uninterrupted functionality when API limits are reached.

## 🚀 Features
- AI-generated interview questions
- Role-based technical interviews
- Answer evaluation and scoring
- Instant feedback and suggestions
- Offline fallback question bank
- Interactive Jupyter Notebook interface

## 🛠️ Tech Stack
- Python
- Google Gemini API
- Jupyter Notebook
- Generative AI

## 📋 How It Works
1. Enter a Job Role and Job Description.
2. Generate an interview question.
3. Submit your answer.
4. Receive a score and feedback instantly.

## 🔮 Future Enhancements
- Voice-based interviews
- Resume-based question generation
- Performance analytics dashboard
- PDF feedback reports
